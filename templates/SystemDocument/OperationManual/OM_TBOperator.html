題庫操作員說明

  • 主要負責輸入紙本題目至系統中，再提交給題庫管理員審核。
    • Question List：試題列表
    • Create Listening：建立聽力試題
    • Create Reading：建立閱讀試題

Question List：試題列表

  • ：題庫操作員提交該試題等待審核。
  • ：題庫操作員編輯該試題，並使該試題進入Saved。
    • 編輯試題僅能更改該試題的「試題類型」及「正確答案」。
    • 更改選項內容持續開發中。
  • ：題庫操作員刪除該試題。

Create Listening：建立聽力試題

  • 「File.」：選擇該試題音檔。
  • 「Type.」：選擇該試題試題類型。
  • 「Difficulty.」：設置該試題難度。difficulty區分為：
    • Easy
    • Medium
    • Hard
  • 「Question Option.」：該試題的４個選項。

Create Reading：建立閱讀試題

  • 「Question Content.」：閱讀試題的題目內容。
  • 其餘欄位請參考建立聽力試題。

多重建立試題

  • 以匯入Excel檔（xls, xlsx）來建立試題。
  • 「File Format.」：規範該檔案需如何輸入資料，題庫操作員需遵守以下資料順序：
    • 試題內容➡️選項１➡️選項２️➡️選項３️➡️選項４️➡️答案（填入1,2,3,4）➡️試題類型（填入1,2,3）➡️難度(填入1,2,3)️
