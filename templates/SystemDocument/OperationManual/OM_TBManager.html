題庫管理員說明

  • 主要負責管理及審核試題

試題列表

  • 表格內主要顯示試題的「序列」、「內容」、「種類」、「狀態」、「其他」
    • 「序列」 : 代表該試題的流水號。
    • 「內容」 : 代表該試題的內容。
    • 「種類」 : 代表該試題的試題類型。
    • 「狀態」 : 代表該試題的狀態。
    • 「其他」 : 代表該試題的功能鍵。
  • 點擊該試題的欄位可顯示該「試題的選項」、「正確答案」、「最新更新時間」、「使用次數」、「難易度」、「創建人」。
  • 試題類型區分為 : 問答、簡短對話、文法、名詞片語、段落理解。
  • 試題狀態區分為 :
    • 暫存 : 該試題在題庫操作員輸入後，會暫時存放在題庫操作員的題目列表中，待題庫操作員確認無誤後即可送出交由題庫管理員進行審核。
    • 通過 : 該試題在通過審核之後即可成為題庫中的試題，將可供考試管理員進行選用。
    • 未通過 : 該試題內容或選項可能有誤，待題庫操作員更改後即可重新提交審核。
    • 等待審核 : 該試題尚未被題庫管理員進行審核。
    • 有問題 : 該試題在被使用後，被受測者回報有問題，包含題目有誤或是選項無法選取等等。
    • 已處理 : 該試題在被回報錯誤後，經由題庫管理員進行處理，而該試題將不被繼續進行使用。
    • 以上「有問題」及「已處理」的問題回報機制持續開發中。
  • 「搜尋」功能列 : 可進行「搜尋」、「種類」、「難易度」及「狀態」。
    • 「搜尋」 : 可透過試題的內容或是試題選項進行試題的搜尋。
    • 「狀態」 : 題庫管理員僅能透過通過、有問題、已處理進行搜尋。

    • 審核試題

    • : 題庫管理員使該試題通過審核。
    • : 題庫管理員編輯該試題並使其通過審核。
      • 題庫管理員編輯功能，僅提供更改答案。
    • : 題庫管理員退回該試題。
      • 題庫管理員在退回時需填寫相關的訊息來說明為何退回？
