A free practice platform for Military Academy Students.

ALCPT is short for American Language Course Placement Test.
ALCPT distinguishes multiple user permissions, and students can focus on their own permissions to practice.
This system will also generate some simple statistical graphs to make it easier for users to understand their practice status.

System Manager. Controlling about system.

The main task of the system manager is to control some personal information of users on the system. Like personal information, student information, etc.

Test Manager. Controlling about exam.

The main task of the test manager is to create and edit test papers, and to create groups of testees. Use these test papers to create mock tests to test specific groups of subjects.

TB Manager. Maintain and review questions

The main task of the TB manager is to review the questions submitted by the question operator and to maintain these questions.

TB Operator. Create new questions.

The main job of the problem operator is to add new questions and submit them to the bank manager for review.

Viewer. Watch the test status of users via this system

This permission is mainly designed for the chief or teacher to watch the testee's practice status through this system and evaluate whether there is a tendency to improve their English ability.

Testee. Main users of this system

Users can practice listening and reading exercises in English through this system. There are also mock tests designed to be performed by administrators. Allows users to use these tests to improve their understanding of English test questions.
